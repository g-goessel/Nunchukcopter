Nunchukcopter
=============

Programme Python pour controller un Hélicoptère avec un raspberry par IR

Dépendances
===========

* Python 2.x (j'aurais aimé python 3.x mais cwiid n'est pas compatible)
* cwiid ( https://github.com/abstrakraft/cwiid )
* Une clée Bluetooth
